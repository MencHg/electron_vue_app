# Electron_vue_app
## Project setup
```
yarn install
```
> 这是一个使用Electron 和@vue/cli 协同开发的Electron 项目
### Compiles and hot-reloads for development
```
yarn serve 
yarn electron:serve
```

### Compiles and minifies for production
```
yarn build
yarn electron:build
```
